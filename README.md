# Aname Branding Design System

A comprehensive design system for **Aname Business Intelligence**, including logos, brand guidelines, and visual assets for use across all design applications and platforms.

## Overview

This repository contains the official branding assets and design system documentation for **Aname Business Intelligence** (abbreviated as "Aname"). It is designed to be imported into Claude Design for seamless design workflow integration. All assets are organized by type and format for easy accessibility and implementation.

**Note:** "Aname" is shorthand for "Aname Business Intelligence" — the complete company name.

## 📦 Contents

### Brand Guidelines
- **BRAND BOARD Y BRAND BOOK/** - Complete brand guidelines document
  - `BRANDBOARD ANAME.pdf` - Comprehensive brand board and brand book with colors, typography, guidelines, and brand voice

### Logo Assets
- **LOGOTIPO/** - All logo variations and formats

#### CURVAS/ (Vector Files)
- Adobe Illustrator source files (.ai) with editable curves and paths
- `Logo final ANAME.ai` - Master logo file with all variations

#### JPEG/ (Raster Format)
- High-quality JPEG versions for various applications
- 28 logo variations for different use cases
- **Logo Aname con Variantes para Registro IMPI/** - Special variants for intellectual property registration
  - Official registration variants (aname-01.jpg through aname-04.jpg)

#### PNG/ (Transparent Format)
- 28 transparent PNG files for digital applications
- Web-ready format with no background
- Optimal for overlay and placement scenarios
- Files: `aname-sinfondo-01.png` through `aname-sinfondo-28.png`

### Stationery & Collateral
- **PAPELERIA/** - Business collateral and stationery assets
  - FONDO DEGRADADO/ - Gradient background templates for branded materials
  - HOJA-MEMBRETADA-aname.ai - Letterhead design file
  - TARJETA PRESENTACION/ - Business card designs and templates
  - MOCKUPS/ - Product mockups and device previews
  - Additional branded stationery templates

### Photography Assets
- **FOTOS/** - Professional brand photography
  - Fotos Aname por Paola Foto Arte/ - 83 professional photographs
  - Curated imagery for web, social media, and marketing materials
  - High-quality brand photography for visual consistency

### Brand Notebooks & Products
- **LIBRETAS/** - Branded notebooks and journal products
  - Physical product designs and templates

### Digital Publications
- **eBooks FINALES/** - Finalized eBook and digital publication assets
  - Ready-to-use digital content for distribution

### Typography
- **TIPOGRAFIAS/** - Brand font library
  - Hero New UltraLight.otf - Light weight typography
  - PointBold.otf - Bold weight typography
  - PointBook.otf - Regular weight typography
  - Taste Of Sweet fonts - Decorative typography options

## 🚀 Getting Started with Claude Design

1. **Clone or download this repository**
   ```bash
   git clone https://github.com/[your-username]/Aname-Branding.git
   ```

2. **Import into Claude Design**
   - Open Claude Design
   - Use the "Import Design System" feature
   - Select the LOGOTIPO folder or individual asset files
   - Reference the brand guidelines in BRAND BOARD Y BRAND BOOK

3. **Access Brand Guidelines**
   - Review `BRANDBOARD ANAME.pdf` for complete brand specifications
   - Follow color palettes, typography rules, and spacing guidelines
   - Maintain consistency with the brand voice and visual identity

## 💡 Best Practices

### Logo Usage
- Always maintain clear space around logos
- Never distort or alter proportions
- Use appropriate format for the intended medium
- Reference BRANDBOARD for specific guidelines on sizing and placement

### Color Consistency
- Refer to BRANDBOARD for official color specifications
- Maintain color accuracy across all implementations
- Use CMYK for print, RGB for digital

### File Formats
- **AI files**: For design system modifications and custom variations
- **PNG files**: Preferred for digital interfaces and web
- **JPEG files**: For print materials and high-quality exports
- **PDF files**: For documentation and reference
